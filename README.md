# Dron_NeforsTeam

## Запуск проекта
Фронтенд и сервер можно запустить с помощью команды `docker-compose up -d`. Для запуска unity можно использовать или же редактор unity или же там же собрать его. 

## Используемые технологии и библиотеки
### Инфраструктура
- Postgresql
- Docker
### Frontend 
- react
- vite
- zustand
- tailwindcss
- material-ui
- lucide-react
### Backend
- fastapi
- websocket
- asyncpg
